## Installation

```shell
npm i
```
## Deploy

```shell
npx hardhat run scripts/deployFactory.ts --network taikoHeklaTest
```

## Verify Contract

```shell
npx hardhat run scripts/verifyFactory.ts --network taikoHeklaTest 
```

## Test

```shell
npx hardhat test
```
